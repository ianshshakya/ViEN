## 🚀 Quick Start

### Prerequisites
- Node.js v16+
- npm v8+

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/vien-chat.git

# Install dependencies
cd vien-chat
npm install

# Start development server
npm start
